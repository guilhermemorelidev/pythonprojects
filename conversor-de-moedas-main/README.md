# 💸 Conversor de Salário (USD para BRL)

Este é um script Python simples que consulta a cotação atual do dólar em tempo real e converte um valor informado (salário) para Reais (BRL), aplicando a formatação monetária brasileira.

## ✨ Funcionalidades

- **Cotação em tempo real:** Utiliza a [AwesomeAPI](https://docs.awesomeapi.com.br) para buscar o valor do par USD-BRL.
- **Formatação Brasileira:** Converte o padrão decimal americano (`1,234.56`) para o padrão brasileiro (`1.234,56`).
- **Interface Limpa:** Limpa o terminal automaticamente antes da execução.

## 🛠️ Tecnologias

- **Python 3**
- **Biblioteca Requests:** Para comunicação com a API de economia.
